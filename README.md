# To-Do List Application (Python CLI)

A simple and efficient **Command-Line To-Do List Application** built using Python.  
This project helps users manage daily tasks with features like adding, viewing, deleting, searching, and filtering tasks, along with persistent storage using a JSON file.



#Features

- Add new tasks (title, description, priority, deadline)  
- View all tasks with status (Completed / Pending)  
- Mark tasks as completed  
- Delete tasks  
- Search tasks by keyword  
- Filter tasks (Completed / Pending)  
- Data stored in JSON file (persistent storage)  
- Exception handling for safe execution  



## 🛠 Technologies Used

- Python 3  
- JSON (for storing data)  
- Built-in libraries:
  - `json`
  - `os`
  - `datetime`
